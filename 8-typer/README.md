# Typer

A typing speed test application that measures your words per minute (WPM) and accuracy.

## Features

- Real-time typing speed calculation
- Accuracy tracking
- Timer functionality
- Word count display
- Error highlighting
- Multiple difficulty levels

## Project Structure

```
Typer/
├── README.md
├── index.html
├── src/
│   ├── js/
│   │   └── index.js
│   └── css/
│       └── style.css
```

## How to Use

1. Open `index.html` in your browser
2. Click start to begin the typing test
3. Type the displayed text as quickly and accurately as possible
4. View your WPM and accuracy at the end
5. Try different difficulty levels

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Keyboard Event API
- Timer functions
